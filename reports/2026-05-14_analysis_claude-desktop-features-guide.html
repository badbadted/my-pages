💬 核心對話功能

英文名稱中文譯名功能說明方案
Conversations 對話 與 Claude 進行文字互動的基礎功能，支援多輪對話、上下文記憶、歷史紀錄搜尋 Free
Starred Chats 星標對話 將重要對話標記星號，方便日後快速找回常用或關鍵對話 Free
File Upload 檔案上傳 上傳文件、圖片、PDF、程式碼等檔案，Claude 可直接讀取並分析內容 Free
Voice Dictation 語音聽寫 按住快捷鍵（Caps Lock）即可語音輸入，即時轉文字送出給 Claude Free
Quick Entry 快速輸入 macOS 雙擊 Option 鍵叫出浮動輸入框，無需切換視窗即可快速提問（顯示最近 5 則對話） Free
Extended Thinking 延伸思考 Claude 展示其推理過程（思考鏈），適用於複雜問題的逐步拆解與邏輯推導 Pro
LaTeX Rendering LaTeX 渲染 在對話中直接渲染數學公式，支援行內與獨立區塊的數學排版 Free

🤖 智慧代理能力

英文名稱中文譯名功能說明方案
Claude Cowork Claude 協作 在隔離 VM 中執行知識工作的代理能力，可直接存取本機檔案，支援長時間背景任務（研究、整理、文件產出），macOS 與 Windows 皆可用 Pro
Computer Use 電腦操控 Claude 可操作你的電腦——移動滑鼠、點擊按鈕、瀏覽網頁、開啟應用程式，當內建工具不足時自動切換至此模式 ProPreview
Remote Control 遠端遙控 透過手機 Claude App 發送指令，桌面版 Claude 自動在電腦上執行操作（需搭配 Computer Use） Pro
Claude Code (Desktop) Claude Code 桌面版 在桌面應用程式中直接使用 Claude Code 的完整開發能力，支援多個本機與遠端 session 平行運作 Max
Claude Design Claude 設計 Anthropic Labs 推出的視覺設計工具，透過對話協作產出設計稿、原型、簡報、文宣等視覺作品，支援行內評論與直接編輯 ProNew
Scheduled Tasks 排程任務 設定定時執行的自動化任務，可用於 PR 監控、部署檢查、定期報告等背景工作 Pro

🔌 擴充與整合

英文名稱中文譯名功能說明方案
Desktop Extensions 桌面擴充套件 安全的本機擴充套件，一鍵連接桌面應用程式與資料（檔案系統、行事曆、郵件、通訊軟體等），使用程式碼簽章與加密儲存 Free
Plugins 外掛 預封裝的功能套件，整合技能、連接器、斜線指令與子代理為單一安裝單位，涵蓋銷售、財務、法律、行銷等職能 Pro
Connectors 連接器 將 Claude 連接到外部服務與資料來源的橋接元件（如 Google Drive、Slack、Notion 等） Pro
Skills 技能 定義 Claude 特定工作流與行為模式的指令集，可自訂或從市集安裝，搭配外掛使用 Pro
MCP (Model Context Protocol) 模型上下文協定 開放標準協定，讓 Claude 以統一介面連接各種工具與資料來源（本機 Server 或遠端服務） Free
Customize Section 自訂中心 設定中的統一管理頁面，集中瀏覽與管理技能、外掛、連接器與排程任務 Free
Plugin Marketplace 外掛市集 統一目錄，在一個地方搜尋、安裝所有可用的技能、連接器與外掛，團隊版支援管理員審核控管 Pro

🎨 創作與視覺化

英文名稱中文譯名功能說明方案
Artifacts 成品面板 Claude 產出的程式碼、文件、圖表、互動內容會在側邊面板即時預覽，可直接編輯、匯出或分享 Free
Live Artifacts 即時成品 成品可連接 MCP Server 取得即時資料（試算表、資料庫、API），從靜態快照升級為動態工具 ProNew
Code Execution 程式碼執行 在對話中直接執行程式碼，產出試算表、簡報、文件等檔案，結果即時呈現在成品面板 Pro
Visualizations 視覺化圖表 在回覆中直接建立互動式圖表、圖形與資料視覺化，支援即時渲染 Free
Canvas 畫布 在成品面板中直接編輯文件或程式碼的互動式編輯區，支援行內修改與 Claude 協同編輯 Free

👤 個人化與記憶

英文名稱中文譯名功能說明方案
Memory 記憶 Claude 自動從過去對話中提取摘要，跨 session 攜帶上下文。可在「設定 → 功能 → 記憶」中檢視與編輯 Free
Projects 專案 持久化工作空間，將檔案、自訂指令、專案記憶集中管理。開啟專案時 Claude 自動載入所有背景脈絡 Free
Project Instructions 專案指令 在專案中設定自訂指令（System Prompt），Claude 在該專案下的所有對話都會遵循這些指令 Free
Profile Instructions 個人偏好設定 在個人帳號層級設定偏好指令，影響所有對話中 Claude 的回應風格與行為 Free
Styles 回應風格 自訂或選擇 Claude 的回應風格（正式、簡潔、詳細等），可建立多組風格並快速切換 Free
Cross-device Sync 跨裝置同步 對話、專案、記憶與偏好設定在所有登入的裝置間自動同步（桌面版、網頁版、手機版） Free

⚙️ 平台與設定

英文名稱中文譯名功能說明方案
macOS App macOS 應用程式 macOS 原生桌面應用程式，支援所有功能（Cowork、Computer Use、擴充套件等） Free
Windows App Windows 應用程式 Windows 原生桌面應用程式，與 macOS 功能完全對等（本機檔案存取、多步驟任務、外掛、MCP） Free
iOS / Android App 手機應用程式 行動裝置版 Claude，支援對話、檔案上傳、語音輸入。可搭配桌面版進行遠端遙控任務指派 Free
Web App (claude.ai) 網頁版 瀏覽器存取的完整 Claude 介面，無需安裝，與桌面版共享所有對話與專案資料 Free

🏢 團隊與企業

英文名稱中文譯名功能說明方案
Enterprise Configuration 企業組態設定 企業管理員可集中管理桌面應用程式的設定，控制功能啟用、安全政策與部署策略 Enterprise
User Groups & Custom Roles 使用者群組與自訂角色 管理員可建立群組（手動或透過 SCIM 自動同步），為每個群組指定自訂角色以控制可使用的 Claude 功能 Enterprise
Admin Plugin Controls 管理員外掛控管 企業管理員可針對組織內的外掛進行審核、限制或封鎖，確保資料安全與合規 Enterprise
SCIM Provisioning SCIM 自動化帳號管理 透過身份識別提供者（IdP）自動同步使用者與群組，支援標準 SCIM 協定 Enterprise

📌 名詞解釋

術語定義
MCPModel Context Protocol（模型上下文協定）— Anthropic 發起的開放標準，讓 AI 模型透過統一介面連接各種工具與資料來源
Artifact成品 — Claude 產出的獨立內容（程式碼、文件、圖表），在側邊面板即時渲染，可匯出為檔案
Cowork協作 — 在隔離虛擬環境中執行的 AI 代理，可長時間背景執行知識工作任務
SCIMSystem for Cross-domain Identity Management — 跨域身分管理系統，用於自動化帳號同步
VMVirtual Machine（虛擬機器）— Cowork 運行的隔離環境，確保安全性
IdPIdentity Provider（身份識別提供者）— 如 Okta、Azure AD，用於企業單一登入
Extended Thinking延伸思考 — Claude 展示推理過程的功能，類似「思考鏈」（Chain of Thought）
Desktop Extension桌面擴充套件 — 安裝在本機的安全擴充，透過 MCP 協定橋接 Claude 與桌面應用程式
