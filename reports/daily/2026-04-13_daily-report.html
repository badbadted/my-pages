術語表
本日報告中出現的關鍵技術術語解釋
Agent Harness（代理框架/線束）
包裹 AI Agent 運行環境的架構層，負責管理工具調用、記憶存取、技能路由、可觀測性等基礎設施，讓 Agent 邏輯與底層執行環境解耦。LangChain 認為這是下一代 Agent 開發的核心概念。
Deep Agents Deploy
LangChain 發布的開源 Agent 部署框架 Beta 版，定位為「模型無關、開源」的替代方案，對標 Anthropic 的 Claude Managed Agents 托管服務。允許開發者在自有基礎設施上部署 Agent。
LlamaParse v2
LlamaIndex 的雲端文件解析服務第 2 版，採用分層配置模型（tiers），相比 v1 成本降低 50%、精度提升。主要用於複雜格式文件（PDF、Word、掃描件）的結構化提取。
LiteParse
LlamaIndex 於 2026-03-19 發布的開源本地文件解析器，零外部依賴，可在本地執行，適合有資料隱私要求的場景。與雲端 LlamaParse 形成互補的雙軌策略。
VLM（視覺語言模型）
Vision Language Model，能同時處理圖像和文字的多模態模型。在文件 OCR 場景中，VLM 被用來「看懂」掃描件或複雜排版，但 LlamaIndex 的研究發現其在生產環境中有特定失效模式需要工程處理。
Project Glasswing
Anthropic 2026-04-07 宣布的聯合安全計畫，集結 AWS、Apple、Broadcom、Cisco、CrowdStrike、Google、JPMorganChase、Linux Foundation、Microsoft、NVIDIA、Palo Alto Networks 共同保護全球關鍵軟體基礎設施的供應鏈安全。
Managed Agents（托管代理）
由 AI 服務商（如 Anthropic）提供完整托管的 Agent 運行環境，開發者只需定義邏輯，底層基礎設施、安全、擴展性由服務商負責。對應概念是自部署的開源框架（如 LangChain 的 Deep Agents Deploy）。
Harness Hill-Climbing（框架爬坡）
LangChain 提出的 Agent 改善方法論，借用機器學習中「爬坡優化」（hill climbing）概念，透過評估（evals）驅動的迭代改善，讓 Agent Harness 持續優化——而非靠人工猜測下一步改進方向。
NVIDIA NemoClaw
NVIDIA 的代理最佳化技術，CrewAI 在 2026-03-17 的文章中展示了與其整合後可實現「自我演化代理」，速度提升 100 倍。NemoClaw 特別針對 AI 代理推理效率進行了硬體層級最佳化。
Eval（評估/驗證）
在 Agent 開發語境中，Eval 指對 Agent 輸出品質的系統性評估機制，包括自動化測試案例、評分器（graders）設計、regression 偵測等。LangSmith 是 LangChain 生態的主要 Eval 工具。
