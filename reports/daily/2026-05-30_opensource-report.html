名詞解釋

本週新出現的術語
Agent Skills
把代理能力封裝成可重用「技能」單元的架構，每個 Skill 含觸發條件與工作流，可組合與目錄化管理。
Self-Evolving Skills（自我進化技能）
SkillOpt 提出，讓代理依執行回饋自動優化、新增或淘汰自身技能，而非靠人手工維護。
Uncensored / 去審查模型
移除官方安全對齊與內容拒答限制的模型版本（如 heretic 工具產出），可回應原版會拒絕的請求，伴隨安全風險。
Video Avatar（音訊驅動數位人）
輸入一段音訊即生成對應口型與表情的虛擬人講話影片（talking-head），如 LongCat-Video-Avatar。
Sparse Autoencoder（SAE，稀疏自編碼器）
可解釋性工具，將模型內部激活拆解為人類可理解的稀疏特徵，用於審計模型行為（如拒答深度）。
World Model（世界模型）
學習環境動態、能預測與生成後續狀態的模型；minWM 為即時互動式影片世界模型的開源框架。
A1B / A3B（Active Billion）
MoE 模型命名後綴，指每次推論「實際激活」的參數量（如 8B-A1B = 總 8B、激活 1B），衡量實際運算成本。
WebGPU 端側推論
透過瀏覽器 WebGPU API 在使用者裝置本地跑模型（如 bonsai-image-webgpu），無需後端伺服器，兼顧隱私與成本。
