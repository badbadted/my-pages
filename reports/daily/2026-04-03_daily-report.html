Architecture

The Convergence of Agent Layers: Harness, Memory, and Document

Across the four sources, three complementary architectural layers are being developed in parallel. LangChain is defining the orchestration harness layer — the middleware-enabled, skill-composable execution environment. CrewAI is advancing the memory and adaptation layer — cognitive memory stores and self-improvement feedback loops. LlamaIndex is building the document intelligence layer — the intake pipeline that converts unstructured enterprise data into agent-ready structured knowledge. Anthropic provides the model and protocol foundation via Claude models and MCP. These four layers are not competing; they are complementary and increasingly interdependent.

Open Models

Threshold Hypothesis: What Changes When Open Models "Cross Over"

LangChain's new post title — "Open Models have crossed a threshold" — implies a capability step-change. Historically, agent frameworks defaulted to GPT-4 or Claude because open-weight models couldn't reliably handle function calling, multi-step reasoning, and tool use. If that threshold has genuinely been crossed (likely referring to models in the 70B+ class, possibly Llama 4 or Qwen-class successors), the downstream consequences are significant: cost drops by 90%+, latency improves for on-device inference, and enterprises can deploy without routing sensitive data through external APIs. This directly threatens the monetization model of closed API providers while validating the broader agent ecosystem.

LangChain's continued emphasis on evaluation infrastructure (Agent Evaluation Readiness Checklist, "How we build evals for Deep Agents", LangSmith CLI for eval pipelines, Evaluating Skills) suggests the field is moving toward a model where agents don't ship without passing structured eval suites. Glossary

Agent Harness
LangChain's term for the structured runtime environment that wraps an AI agent — composed of anatomy (core loop), middleware (customization hooks), and skills (capability modules).
LangSmith Fleet
LangChain's platform for managing, monitoring, and deploying fleets of production agents at scale. Analogous to a fleet management system for autonomous vehicles.
Deep Agents
LangChain's framing for agents that operate over extended time horizons with multiple tool calls and multi-step reasoning, as opposed to single-turn or shallow agents.
LangSmith Sandboxes
Isolated execution environments within LangSmith that allow safe testing of agent behaviors before production deployment.
MCP (Model Context Protocol)
Anthropic-originated open protocol for structured communication between AI models and external tools/data sources. Donated to the Agentic AI Foundation in December 2025 for open governance.
Claude Agent SDK
Anthropic's SDK for building agentic applications with Claude models, with native support for tool use, multi-turn reasoning, and computer use via the Vercept acquisition.
LlamaParse
LlamaIndex's document parsing service, now in v2. Specializes in accurately extracting structured data from complex PDFs, tables, and multi-format documents at scale.
LiteParse
LlamaIndex's open-source, locally-runnable document parsing library. Enables on-premises document processing without sending data to external APIs.
LlamaAgents Builder
LlamaIndex's natural-language-driven agent creation tool. Users describe an agent in plain language; the builder generates and deploys the agent pipeline.
Agentic OCR
LlamaIndex's approach to OCR that treats document recognition as an agent task with verification and retry loops, achieving 90–95%+ straight-through processing rates.
CrewAI AMP
CrewAI's Agent Management Platform — the enterprise offering that provides deployment, monitoring, and governance infrastructure for CrewAI-orchestrated agent workflows.
Cognitive Memory
CrewAI's architecture for giving agents human-like memory: episodic (what happened), semantic (what is known), and procedural (how to do things) memory stores that persist across sessions.
NemoClaw (NVIDIA)
NVIDIA's framework for self-evolving AI agents. Integrated with CrewAI to enable agents that improve their own strategies iteratively, with reported 100x throughput improvement over baseline.
Open Models Threshold
LangChain's framing for the capability point at which open-weight language models become reliably useful for production agent workloads — enabling self-hosted, cost-efficient agent deployment.
OmniDocBench
A benchmark suite for evaluating document parsing accuracy. LlamaIndex achieved 94%+ accuracy and noted the benchmark is approaching saturation, signaling a need for harder evaluation standards.
Agentic AI Foundation
The open governance body established (December 2025) to steward the Model Context Protocol and related agentic AI standards, following Anthropic's donation of MCP.
