🧠 深度分析

洞察 1：Anthropic 的雙軌擴張

一方面以 Mythos 模型切入資安垂直市場（攻），另一方面透過 Project Glasswing 建立跨廠商安全聯盟（守）。同步推進 MCP 捐贈給 Agentic AI Foundation，顯示其試圖主導 Agent 生態的標準制定層，而非僅做模型供應商。$3,800 億估值背後是年化 $140 億 + 10x 成長的財務支撐。

洞察 2：Agent 評估成為下一個競爭戰場

LangChain 在短期內密集發表「Evaluating Skills」、「Agent Evaluation Readiness Checklist」、「How we build evals for Deep Agents」，並將 LangSmith 定位為評估標準工具。當 Agent 能力趨於同質化，「如何客觀衡量、持續監控 Agent 表現」將成為企業採購決策的關鍵差異點。

洞察 3：文件 AI 正從 RAG 走向 Agentic

LlamaIndex 的技術演進軌跡清晰：從 RAG 框架 → LlamaParse 精準解析 → Agentic OCR（自我校正、跨頁追蹤）→ Long Horizon Document Agents（週級自主任務）。VLM 作為 OCR 後端已進入生產驗證階段，但失效模式研究才剛開始，這是接下來 3-6 個月的技術熱點。

📚 詞彙表
Agent Harness（代理框架）
LangChain 提出的三層 Agent 架構：Skills（技能層）、Middleware（中介層）、Orchestration（編排層），提供可插拔的代理行為客製化機制。
LangSmith Fleet
LangChain 的企業級代理管理平台，提供 Skills 共享、多代理部署、評估監控與 Arcade.dev 工具整合。
VLM-Powered OCR（視覺語言模型驅動 OCR）
以視覺語言模型（如 GPT-4V、Claude Vision）替代傳統 OCR 引擎，可理解語意與版面，但在複雜表格、多語言混合場景有特定失效模式。
Cognitive Memory（認知記憶）
CrewAI 提出的代理記憶架構，實現跨任務、跨 Session 的記憶持久化與情境感知，比傳統向量記憶庫更接近人類工作記憶模式。
MCP（Model Context Protocol）
Anthropic 開發並已捐贈給 Agentic AI Foundation 的開放協定，標準化 LLM 與外部工具/資料來源的互動介面，成為 Agent 生態的基礎設施層。
