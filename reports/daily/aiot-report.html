ESPHome 2026.4.5 — 感測器發布速度 46 倍提升 + OTA 簽名安全
⏱ 1 小時
2026.4.0 的核心改進：感測器資料發布速度最高提升 46 倍（log 格式化從裝置端移至客戶端），ESP32 預設 240MHz（API 操作快 33%）。OTA 簽名驗證讓韌體更新更安全，不需要硬體 Secure Boot。GPIO 擴展器中斷驅動使 I2C 閒置流量減少 99.7%——直接影響 FP2/Zigbee 感測器的即時響應精度。

⏱ 1–2 小時

操作步驟
  1. 在 192.168.0.229 更新 llama.cpp：git pull && cmake -DGGML_CUDA=ON -B build && cmake --build build -j --config Release
  2. 下載 Qwen3.5-14B-Instruct GGUF（Q4_K_M）：huggingface-cli download Qwen/Qwen3.5-14B-Instruct-GGUF
