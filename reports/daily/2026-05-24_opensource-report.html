本週重大亮點

跨來源綜合分析，最值得關注的 7 個訊號
1
Agent Skills 生態全面爆發 — 從框架到技能包的典範轉移
GitHub 前 10 出現 4 個 Skills 倉庫（academic-research-skills #1 +11,550、scientific-agent-skills #5、anthropics/skills #6、claude-plugins-official #10），加上 HKUDS/CLI-Anything（CLI 全代理化 #7），標誌著 AI 應用開發從「搭模型框架」轉向「組裝 Skill 技能包」的典範轉移。MCP 生態學術化（arXiv HarnessAPI Skill-First 框架）也同步印證這個方向。
2
ByteDance Lance 與 SenseNova-U1 — Any-to-Any 統一多模態大戰開打
ByteDance Research/Lance 空降 HF 模型 #1（Any-to-Any，整合理解/生成/編輯），HF Papers 同步上榜（71票 #4）。SenseNova-U1（185票 #2）以 NEO-unify 架構統一多模態，Meta SAM 3（137票 #3）推出概念分割。三家在同一週競相展示統一多模態路線，Any-to-Any 架構霸主之爭正式開打。
3
文件 VLM 三強同週 — 緊湊型文件理解成最熱研究聚落
HF Papers 同週出現 MinerU2.5（1.2B，162票）、SmolDocling（256M，159票）、PaddleOCR-VL（0.9B，126票）三篇文件解析 VLM，分別來自大型研究機構、IBM Granite 和百度 PaddlePaddle。文件理解 AI 的競爭重點從「最強」轉向「最小且最準」，256M 就能做文件轉換的 SmolDocling 代表這個趨勢的極致。
4
DeepSeek-V4-Pro 下載三週 12 倍 — 產業採用已進入無競爭者區間
DeepSeek-V4-Pro 本週達 4.51M 下載（上週 2.97M，+52%），三週內從 382k 累計成長 12 倍（4/24→5/24）。同週 likes 4,190，是 HF 前 15 模型中遠超第二名（circlestone/Anima 1,510）的壓倒性領先。開源 LLM 採用的馬太效應，DeepSeek 已進入無人能望項背的區間。
5
HKUDS 雙榜 — 香港大學資料科學組成本週最高密度入榜機構
HKUDS（Hong Kong University Data Science）同週兩個專案入 GitHub Python 趨勢：ViMax（代理影片生成，#3，+2,685）和 CLI-Anything（將任何 CLI 工具代理原生化，#7，+4,832）。ViMax 模擬導演/編劇/製片人多角色協作生成影片，CLI-Anything 讓所有現有軟體立即變成 Agent 可用工具，兩個方向均對 AI 實際部署具高度影響力。
6
LLM 能力悖論研究 — 「更強 LLM 預測更差」挑戰 Scaling Law 信仰
arXiv cs.AI 本週出現「Is Capability a Liability? More Capable Language Models Make Worse Forecasts」（2605.22672）——更有能力的 LLM 在預測任務反而表現更差。同週 cs.AI 另有「Can AI Make Conflicts Worse?」研究 LLM 部署的對齊失敗，加上「Accumulated Message Effects on LLM Judgments（AMEL）」，學術界對 LLM 能力的批判性審視全面升溫。
7
後訓練理論統一 — 「狀態而非詞元」視角整合 SFT/RL/蒸餾
arXiv cs.LG 的「Post-Training is About States, Not Tokens」（2605.22731）用狀態分佈視角統一解釋 SFT、RL 和 On-Policy Distillation 三種後訓練方法，同週另有 Clipping Bottleneck（RLVR 穩定化）和 Vector Policy Optimization（多樣性策略），LLM 後訓練理論正在經歷快速統一化，實務影響深遠。
