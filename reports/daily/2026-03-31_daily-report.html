Deep Analysis — 深度分析

🏗️ 技術架構演進：Agent 從「跑起來」到「跑得好」
Evaluation-First 設計哲學興起：LangChain 以 17 分鐘長文「Agent Evaluation Readiness Checklist」作為精選置頂，並有多篇 Eval 相關文章，顯示業界正從「快速 prototype」轉向「可評估、可量測的生產 agent」。未來 agent 系統的競爭力將部分取決於 eval 基礎設施的成熟度。
三層 Harness 架構（LangChain）vs 認知記憶架構（CrewAI）：兩個框架從不同角度解決同一問題——讓 agent 更可靠、更持久。LangChain 以 Harness（Core Logic + Middleware + Skills）作為結構化部署框架；CrewAI 以 Cognitive Memory 解決跨任務的狀態持久性。兩者互補而非競爭，企業可根據需求選擇側重點。
Self-Evolving Agents 是下一個關鍵能力邊界：CrewAI + NVIDIA NemoClaw 的「自我進化」概念若成熟，將從根本改變 agent 維護模式——從人工 prompt tuning 轉向 agent 自我優化。100x 速度提升的聲稱需要獨立驗證，但方向上代表重要的架構轉變。
MCP 生態成熟度加速：Anthropic 將 Model Context Protocol 捐贈並建立 Agentic AI Foundation，Apple Xcode 原生支援 Claude Agent SDK。MCP 從 Anthropic 的私有協議演變為業界標準的路徑已相當清晰，開發者現在部署 MCP tools 可以預期更廣泛的跨平台支援。
📄 文件處理技術趨勢：OCR → Agentic OCR → Document Automation Platform
LlamaIndex 的平台化策略：一週年後 LlamaParse 宣告升級為「完整文件自動化平台」，這是典型的 SaaS 擴展路徑——從單一解析工具擴展到 Extract → Sheets → Agents 全鏈路，黏住企業客戶。LiteParse 開源版則維持社群入口。
Word 表格解析的企業重要性：今日新文章「Improving Table Parsing for Word (.docx) Documents」看似技術細節，實際上直接影響大量企業文件自動化場景——財務報表、合約、標準化表格在企業環境中大量以 .docx 格式存在，此改進的商業價值被低估。
OCR → Agentic OCR 的能力跳躍：傳統 OCR 直通率通常 70-80%，需要大量人工修正；LlamaIndex 宣稱 Agentic OCR 達到 90-95%+ 直通率，法律文件場景的「理解語義」能力是關鍵突破點。Receipts、Invoices 的自動化對中小企業財務流程有直接降本價值。
💼 商業應用與企業採用加速
四大顧問公司採用 CrewAI 是重要市場訊號：PwC 選擇 CrewAI 作為「Global Agent OS」，這不只是一個案例，而是整個企業諮詢產業開始用 multi-agent 重構服務交付的訊號。四大顧問公司的採用通常帶動更廣泛的企業客戶跟進。
Anthropic 的雙軌策略：模型 + 生態：$30B Series G（估值 $380B）+ $100M Claude Partner Network 的組合，說明 Anthropic 不只是在賣 API，而是在建立類似 AWS Partner Network 的企業生態系。Claude Agent SDK 與 MCP 捐贈是這個策略的技術基礎設施。
Boeing 子公司 Jeppesen 省 2,000 工程小時：這個具體數字是 LlamaIndex 目前最有說服力的 ROI 案例。航空業的嚴格合規要求讓此案例格外有說服力——如果 Agentic Document Processing 能在航空業達標，幾乎任何行業都適用。

Glossary — 技術名詞說明

18 個詞條
Agent Harness
LangChain 提出的 agent 部署架構框架，包含三層：Core Logic（核心推理邏輯）、Middleware（可插拔中介層，處理 auth/logging/retry 等橫切關注點）、Skills（可重用工具能力集合）。類比硬體的「測試治具」，讓 agent 可以被標準化測試與部署。
Agent Evaluation (Eval)
評估 agent 行為正確性的方法論，分為 Offline Eval（用固定測試集評估，快但覆蓋有限）與 Online Eval（生產環境實時評估，真實但成本高）。Grader 是負責判斷 agent 輸出是否正確的評分機制，可以是 rule-based 或 LLM-based。
LangSmith Fleet
LangChain 推出的 agent 車隊管理平台，讓企業能夠集中管理、監控、部署多個 agent 實例。類比 Kubernetes 對容器的管理，Fleet 對 agent 做同樣的事：調度、觀測、滾動更新。
LangSmith Sandboxes
安全的 agent 測試沙箱環境，讓開發者可以在隔離環境中測試 agent 行為，不影響生產系統。對於需要執行程式碼、呼叫外部 API 的 agent 尤其重要。
LlamaParse
LlamaIndex 的雲端文件解析服務，支援 50+ 格式（PDF、Word、Excel、HTML 等）。v2 版本降價 50%，準確率達 94%+。一週年後定位升級為「完整文件自動化平台」，整合 Extract → Sheets → Agents 全鏈路。
LiteParse
LlamaIndex 的開源本地文件解析方案，相對於雲端版 LlamaParse，適用於隱私敏感場景（如醫療、法律、金融），不需將文件上傳到第三方服務。
Agentic OCR
結合 AI Agent 推理能力的光學字元辨識（OCR），相對於傳統 OCR 的像素級字元辨識，Agentic OCR 能理解文件語義、處理複雜版面（表格、多欄、掃描品質差的文件），直通率從傳統 70-80% 提升到 90-95%+。
Cognitive Memory
CrewAI 提出的 agent 記憶機制，讓 agent 能在跨任務、跨 session 的執行中保持狀態與學習。不同於短期的 context window 記憶，Cognitive Memory 是長期的、結構化的知識存儲，解決長期自主 agent 的狀態一致性問題。
Self-Evolving Agents
