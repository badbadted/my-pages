🔱 /fork 與相關指令詳解

/fork 是 Claude Code 的對話分叉機制，搭配 /branch、/resume、/rewind 構成完整的對話版本控制系統。

指令中文行為適用場景
/fork <directive> 分叉子代理 生成一個背景子代理，繼承完整對話歷史，在背景執行指定任務。你的對話繼續不中斷，子代理完成後結果會回傳到你的對話中 想讓 Claude 同時嘗試另一個方向，但自己不想切走
/branch [name] 對話分支 在當前位置建立對話分支，你自己切進新分支。原對話保留，可用 /resume 回去。v2.1.77 前叫 /fork 想親自嘗試不同方向，但保留回退點
/resume [session] 恢復對話 回到之前的對話（含 /branch 和 /clear 留下的）。開啟選擇器或直接指定 session ID/名稱。背景 session 也會出現在列表中 切回原本的對話分支、恢復之前的 session
/rewind 倒帶 / 回滾 把對話和程式碼回滾到先前某個檢查點，或摘要部分對話。類似 git 的 checkpoint + revert 嘗試失敗了，想整個退回到之前某個好的狀態
/clear [name] 清空對話 開始全新對話（清空上下文），舊對話可在 /resume 找到。可為舊對話命名方便辨認。想保留上下文但釋放空間，用 /compact 換一個完全不同的任務

🔑 /fork vs /branch 一句話區分

/fork = 「Claude 去背景跑，我繼續做我的」（非同步、子代理）

/branch = 「我自己切進去試，原本的存著」（同步、自己操作）

注意：v2.1.161 之前 /fork 等同 /branch。之後兩者分開。

📋 上下文管理（Context Management）

指令中文說明別名
/compact [instructions]壓縮對話摘要目前的對話以釋放上下文空間。可指定摘要重點方向
/clear [name]清空對話開始全新對話，舊對話可用 /resume 找回。可為舊對話命名/reset /new
/context [all]上下文視覺化以彩色格線顯示當前上下文使用狀況，含最佳化建議與容量警告。加 all 展開詳細
/branch [name]對話分支建立當前對話的分支副本，切進新分支，原對話保留
/fork <directive>分叉子代理生成背景子代理繼承對話，執行指令後回傳結果。v2.1.161+
/resume [session]恢復對話恢復之前的對話，或開啟 session 選擇器。支援背景 session/continue
/rewind倒帶 / 回滾回滾對話與程式碼到先前的檢查點，或摘要指定訊息/checkpoint /undo
/btw <question>順帶一問問一個快速側面問題，不加入對話歷史（不佔上下文）
/recap摘要回顧產生當前 session 的一行摘要。離開後回來也有自動摘要
/rename [name]重新命名為當前 session 命名，顯示在提示列上。無參數則自動產生

🧠 模型與推理（Model & Reasoning）

指令中文說明別名
/model [model]切換模型切換 AI 模型並儲存為預設。支援左右鍵調整推理強度，按 s 只切當次 session
/effort [level|auto]推理強度設定推理等級：low / medium / high / xhigh / max / ultracode。auto 回復預設。無參數開滑桿
/fast [on|off]快速模式切換快速模式，犧牲深度推理換取更快回應

/ultraplan <prompt>超級計畫在雲端開啟深度計畫 session，可在瀏覽器審閱後遠端執行或拉回本機
/goal [condition|clear]設定目標設定持續目標，Claude 跨回合持續工作直到條件達成。clear 可取消

📝 專有名詞解釋

術語定義
Slash Command斜線指令 — 以 / 開頭的 Claude Code 內建指令，在訊息開頭輸入觸發
Skill技能 — 以 SKILL.md 定義的擴充指令，只在被使用時載入。Claude 可自動判斷何時需要
Workflow工作流程 — 動態編排多個子代理平行執行的進階 Skill，在背景運行
Subagent子代理 — Claude 委派獨立子任務的背景代理，有自己的上下文窗口
Worktree工作樹 — Git 的獨立工作副本，讓多個子代理在不同分支同時改程式碼
Context Window上下文窗口 — Claude 一次能記住的對話與檔案內容總量，壓縮（/compact）可釋放空間
Compaction壓縮 — 將長對話摘要為精華，釋放上下文空間同時保留重要資訊
Checkpoint檢查點 — 對話與程式碼的快照，/rewind 可回滾到此處
Background Agent背景代理 — 脫離終端在背景獨立執行的 Claude session，可用 claude agents 監控
MCP Server模型上下文協定伺服器 — 透過 MCP 協定連接外部工具與資料來源的服務
Routine例行程序 — 在 Anthropic 雲端按排程自動執行的定時任務
Ultracode超級模式 — 結合 xhigh 推理強度與自動 Workflow 編排的最高等級設定
