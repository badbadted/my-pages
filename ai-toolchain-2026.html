① 我目前在用的三套 AI

🤖 Claude Code Max 5x · $100 / 月
主控台 · 調度官 · 中文與安全邊界守門員
  • 角色：對話入口、ACE 任務調度、Skill 編排、UI 視覺迴圈、中文文案、PRD、Markdown 文件
  • 模型：Claude Opus 4.6（1M context）為主、Sonnet 4.6 次選
  • 付費：Claude Max 5x $100/月（Pro $20 的 5 倍配額）
  • 殺手鐧：內建 hooks 機制（PreToolUse/PostToolUse/UserPromptSubmit）可程式化攔截工具呼叫，是目前唯一有「強制執行層」的 AI Agent CLI
  • 使用形態：CLI（互動式對話）+ MCP server（plugin 擴充）+ 桌面/網頁/IDE 多端同步
⚡ OpenAI Codex（via ChatGPT Plus） Plus · $20 / 月
純邏輯跑腿小弟 · 第二雙眼睛
  • 角色：API route / server action、SQL/schema、Python 腳本、單元/整合測試、純技術 debug、refactor
  • 模型：GPT-5.4 為主、GPT-5.4-mini 省額度
  • 付費：ChatGPT Plus $20/月（與一般 ChatGPT 訂閱共用）
  • 配額機制：5 小時滾動視窗，Plus 約 33–168 訊息/視窗（依任務複雜度浮動）
  • 整合方式：已裝 openai/codex-plugin-cc，可從 Claude Code 內呼叫 /codex:rescue、/codex:review、/codex:adversarial-review
  • 使用形態：互動 TUI（獨立）+ Claude Code 內 plugin（背景任務）
🔍 Google Gemini（via API key） 按量計費 · 變動
研究員 · 搜尋引擎 · 圖片生成器
  • 角色：主題研究（gemini-deep-research / gemini-search）、URL 抓取分析、圖片生成、PDF/影片摘要、雙腦交叉驗證
  • 模型：Gemini 2.5 Pro / 3.1 Pro（看 MCP server 配置）
  • 付費：API key 按量，2.5 Pro $1.25/$10 per 1M tokens（input/output），3.1 Pro Preview $2/$12
  • 整合方式：以 MCP server 形式掛在 Claude Code 內（@rlabs-inc/gemini-mcp），提供 30+ tool calls
  • 使用形態：純 MCP 工具，不直接對話，由 Claude 代為呼叫
⚠️ Claude 與 Gemini 的方案/付費若與實際不符請告知，會更新此報告。

⑥ 結論與建議

目前的三軌方案已經接近最佳化——Claude 主控 + Codex 跑腿 + Gemini 做研究與多模態。月固定 ~$120 換來：

  • ✅ 撞上限焦慮中等（Max 5x 多 session 平行有時會撞，但比 Pro 寬鬆）
  • ✅ 兩家模型互審（Claude 寫 + Codex review，反之亦然）
  • ✅ Gemini 補上 Claude/Codex 不擅長的多模態與 deep research
  • ✅ Claude Code 的 hooks 提供唯一的程式化安全層

什麼情況下要換

  • 📉 月用量持續低於 30% Claude 配額 → 降到 Claude Pro ($20)
  • 📈 經常撞 Max 5x 上限 → 升 Max 20x ($200)
  • 🔒 接到不能上雲的客戶案 → 加 Ollama + 本地模型走 Aider
  • 💸 預算砍半 → 改 Claude Pro ($20) + Cursor ($20) + DeepSeek API ($5–10)，總共 ~$50
  • 🏢 接 enterprise 案需要 audit log → 升 Claude Team / Business plan
